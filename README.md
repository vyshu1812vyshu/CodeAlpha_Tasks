# CodeAlpha Python Programming Internship 🚀

Welcome to my official repository for the **CodeAlpha Python Programming Internship**. This repository contains  assigned projects completed successfully as part of the internship.

## 📁 Repository Structure

### 🎮 Hangman Game (`hangman.py`)
A classic text-based word guessing game where players attempt to guess a hidden word within a limited number of attempts.

**Features:**
- Random word selection
- Limited attempts
- User-friendly console interface
- Input validation

### 🤖 Simple Chatbot (`chatbot.py`)
A rule-based chatbot that responds to predefined user inputs and simulates basic conversational interactions.

**Features:**
- Interactive text conversation
- Multiple predefined responses
- Exit command support
- Beginner-friendly AI logic

### 📈 Stock Portfolio Tracker (`portfolio.py`)
A portfolio management application that calculates the total value of a user's stock investments based on stock quantities and prices.

**Features:**
- Tracks multiple stocks
- Calculates portfolio value
- Displays investment summary
